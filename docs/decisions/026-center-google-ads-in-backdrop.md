# ADR 026: Backdrop 상단 광고를 티스토리 수익 치환자 중앙 정렬 슬롯으로 통일

- 상태: 승인
- 날짜: 2026-03-12

## 배경

`backdrop` 상단 무대는 홈/목록/글 상세에서 공통으로 보이지만, 수동 AdSense 삽입은 티스토리 실환경에서 응답 실패 가능성이 있었다.  
수익 광고는 티스토리 치환자를 사용하는 편이 실제 노출과 운영 측면에서 더 안정적이다.

## 결정

- `src/pages/main.html`의 `backdrop-inner` 내부에 티스토리 수익 치환자 `[##_revenue_list_upper_##]`를 직접 배치한다.
- `src/style/style.css`에서 치환자 출력 영역이 중앙 배너형 컨테이너와 최소 높이를 유지하도록 정렬 규칙을 부여한다.

## 결과

- 티스토리 스킨이 제공하는 수익 치환자 경로를 그대로 사용하게 된다.
- 광고 응답 전후에도 상단 무대의 정렬과 높이가 안정적으로 유지된다.
- 수동 AdSense 호출로 인한 실환경 오류 가능성을 줄인다.
