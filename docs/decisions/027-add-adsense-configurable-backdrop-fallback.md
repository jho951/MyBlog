# ADR 027: 홈 상단 광고는 pageHome 상단 section의 티스토리 수익 치환자만 사용한다

- 상태: 승인
- 날짜: 2026-03-13

## 배경

현재 스킨에는 `backdrop` 상단 광고 정렬은 준비되어 있었지만, 해당 위치는 헤더 및 상단 무대 레이아웃과 충돌하기 쉬워 광고가 겹치거나 예상과 다르게 보일 수 있었다.
또한 구글 광고 관련 JS를 별도로 넣지 않고 티스토리 치환자만 사용하려는 운영 요구가 있다.

## 결정

- 메인 홈에서는 `backdrop` 안 광고를 제거하고, `#pageHome` 시작 지점에 `home-revenue-section`을 둔 뒤 `[##_revenue_list_upper_##]`만 출력한다.
- `home-revenue-section`은 광고 주변에 별도 박스 스타일을 두지 않고 위치용 래퍼로만 사용한다.
- 본문 페이지(`pageArticle`)에서는 수동 하단 광고 `#ttMidAd`를 숨겨 자동 광고와 중복되지 않게 한다.

## 결과

- 구글 광고 관련 JS를 스킨에서 직접 관리하지 않아 운영 복잡도를 줄인다.
- 메인 상단 광고는 `pageHome` 내부에서 안정적으로 렌더링되고, 헤더 겹침 문제를 줄인다.
- 본문 페이지에서는 수동 광고 중복이 제거되어 구글 광고가 1개만 보이게 된다.
