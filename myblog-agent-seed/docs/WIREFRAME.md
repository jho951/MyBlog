```text
┌─────────────────────────────┐
│ [Logo]              [☰][🔍] │  ← 캡슐 헤더

┌─────────────────────────────┐
│ HERO                        │

└─────────────────────────────┘

┌─────────────────────────────┐
│ Quick Links                 │
│ [카테고리] [전체 글] [구독]  │  ← 가로 스크롤 칩/카드
│ [방명록] [GitHub] [소개]     │
└─────────────────────────────┘

┌─────────────────────────────┐
│ Featured Content            │
│ [대표 글 카드 1]            │
│ [대표 글 카드 2]            │
│ [대표 글 카드 3]            │
└─────────────────────────────┘

┌─────────────────────────────┐
│ About & Tech Snapshot       │
│ 소개 문단 (짧게)            │
│ 강점 3개 카드               │
│ Backend / FE / Infra tags   │
└─────────────────────────────┘

┌─────────────────────────────┐
│ Category Hub                │
│ [CS 카드]                   │

│ [Development 카드]          │
```

### 모바일 메모

> Hero → Quick Links → Featured 순으로 빠른 탐색 유도
>
> Category Hub는 세로 스택
>
> 카드 정보 밀도 낮게 유지 (제목 + 1~2줄 요약)


## PC 와이어프레임 (>= 1024px)
```
┌──────────────────────────────────────────────────────────────┐
│ [Logo]   [Category] [Series] [About] [Guestbook]   [Search] │
└──────────────────────────────────────────────────────────────┘

┌───────────────────────────────┬──────────────────────────────┐
│ HERO / INTRO                  │ Quick Links / Profile Card   │
│ "해학적 Just do IT"           │ [GitHub] [전체 글] [구독]    │
│ 백엔드/CS/설계 중심 기술블로그  │ 기술 태그 / 핵심 링크         │
│ 설명 + CTA 버튼                │ 방문자용 빠른 진입            │
└───────────────────────────────┴──────────────────────────────┘

┌──────────────────────────────────────────────────────────────┐
│ Featured Content (대표 글/시리즈)                             │
│ [카드1] [카드2] [카드3] [카드4]                              │
└──────────────────────────────────────────────────────────────┘

┌───────────────────────────────┬──────────────────────────────┐
│ About                         │ Tech Snapshot                │
│ 소개 문단 / 강점 / 운영 철학   │ Backend / FE / Infra / Tool  │
│                               │ 스택 카드/태그                │
└───────────────────────────────┴──────────────────────────────┘

┌──────────────────────────────────────────────────────────────┐
│ Category Hub (상위 카테고리 그리드)                           │
│ [CS] [Development] [Engineering] [TroubleShooting]           │
│ 각 카드에 하위 카테고리/글수/설명                             │
└──────────────────────────────────────────────────────────────┘

┌───────────────────────────────┬──────────────────────────────┐
│ Recent Posts                  │ Subscribe / Guestbook Card   │
│ 최신 글 목록                  │ 구독 CTA / 방명록 / Contact  │
│ [전체 글 보기]                │                              │
└───────────────────────────────┴──────────────────────────────┘

┌──────────────────────────────────────────────────────────────┐
│ Footer                                                       │
└──────────────────────────────────────────────────────────────┘
```

### PC 메모

> Hero + Quick Links를 2컬럼으로 배치
>
> Featured / Category Hub 중심으로 허브 역할 강화
>
> Recent Posts와 Subscribe 카드를 분리해 시선 정리


### Tablet 처리 원칙
> PC 2컬럼 섹션 → 1컬럼 또는 2:1 비율로 축소
>
> Category Hub는 2열 카드
>
> Quick Links는 가로 스크롤 또는 2줄 칩 배치
