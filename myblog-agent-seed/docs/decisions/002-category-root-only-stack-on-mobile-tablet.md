# ADR 002: 모바일/태블릿 카테고리 노출을 루트 카테고리 세로 스택으로 통일

- 상태: 승인
- 날짜: 2026-02-28

## 배경

- 홈 카테고리 영역은 기존에 모바일(`catHub--mobile`)과 태블릿/PC(`catStack`)가 분리되어 있었다.
- 모바일에서는 하위 카테고리 카드까지 함께 노출되어 정보량이 과해지고, 태블릿/PC와 탐색 경험이 달라 일관성이 떨어졌다.

## 결정

- 모바일과 태블릿에서도 PC와 동일하게 루트 카테고리만 노출한다.
- 카테고리 영역 레이아웃은 뷰포트 전 구간에서 세로 스택(`catStack`)을 기본으로 사용한다.
- 모바일 전용 카테고리 허브(`catHub--mobile`)는 노출하지 않는다.

## 결과

- 화면 크기와 무관하게 카테고리 진입 구조가 동일해져 학습 비용이 줄어든다.
- 모바일에서 하위 카테고리 과다 노출로 인한 스크롤/시선 분산이 줄어든다.
- 데스크톱의 2열 카드 레이아웃은 기존 미디어쿼리 정책(1025px 이상)대로 유지한다.
