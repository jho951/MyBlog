# ADR 003: 최신글 슬라이더에 pagesNav 카드 스타일 재사용

- 상태: 승인
- 날짜: 2026-02-28

## 배경

- 홈에는 `latest`(최신글)와 `latest--pagesNav`(페이지 네비게이션) 두 카드 스타일이 공존했다.
- 최신글 카드의 시각 언어를 pagesNav 카드와 통일해 홈 카드 일관성을 높일 필요가 있었다.

## 결정

- 최신글 섹션(`#ttLatestFeed`)에 `latest--pagesNav` 클래스를 적용해 카드 스타일을 재사용한다.
- 단, 최신글 섹션의 레이아웃 폭/배경은 기존을 유지하기 위해 `#ttLatestFeed.latest--pagesNav`에 컨테이너 리셋 스타일을 둔다.

## 결과

- 최신글 카드가 pagesNav 카드와 동일한 카드 비주얼(종이 질감/타이포/메타 처리)을 사용한다.
- 기존 최신글 섹션의 배치 흐름과 pagesNav 슬라이더 동작은 유지된다.
