# ADR 004: 홈 페이지 카드 섹션 제거 및 목록 탭 스크립트 외부화

- 상태: 승인
- 날짜: 2026-02-28

## 배경

- 홈의 페이지 바로가기 카드 섹션(`ttPagesNav`)은 현재 탐색 구조에서 우선순위가 낮고, 기능 제거 요청이 있었다.
- `main.html` 하단의 인라인 스크립트는 유지보수와 재사용 측면에서 분리가 필요했다.

## 결정

- 홈의 `ttPagesNav` 섹션(카드 영역 + 메뉴 소스)을 삭제한다.
- `pages-nav.js`는 HTML에서 로드하지 않고, `boot-init.js`의 `initPagesNavSlider` 호출도 제거한다.
- 목록 카테고리/하위카테고리 활성화 로직은 `src/function/list-subcategory-tabs.js`로 분리하고 HTML에서 외부 스크립트로 로드한다.

## 결과

- 페이지 카드 기능/영역이 홈에서 완전히 제거된다.
- `main.html` 하단 인라인 스크립트가 사라져 템플릿 구조가 단순해진다.
- 목록 탭 로직은 독립 JS 파일로 관리되어 수정 범위가 명확해진다.
