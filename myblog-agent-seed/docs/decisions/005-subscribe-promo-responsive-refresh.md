# ADR 005: 구독 프로모션 섹션 반응형 리디자인

- 상태: 승인
- 날짜: 2026-02-28

## 배경

- 홈의 `subscribePromo` 섹션은 모바일/태블릿에서 텍스트와 목업 카드의 밀도가 높아 가독성과 영역 균형이 떨어졌다.
- CTA 버튼의 시각적 강조와 터치 영역도 화면 크기에 따라 일관성이 부족했다.

## 결정

- `subscribePromo`를 모바일 우선으로 재설계하고, 태블릿/데스크톱에서 2열 레이아웃으로 확장한다.
- 섹션 내부에 통합 컨테이너(배경/테두리/그림자)를 적용해 독립된 블록으로 보이게 한다.
- 모바일에서는 텍스트/CTA를 먼저 배치하고 목업 카드는 뒤에 배치한다.
- `btn_subscription`의 폭/높이/라운드를 반응형으로 조정해 터치 접근성과 시인성을 높인다.
- 버튼 클래스의 `#subscribe` 토큰은 제거한다.

## 결과

- 구독 섹션의 정보 계층(라벨/헤드라인/CTA/목업)이 화면 크기별로 명확해진다.
- 모바일에서 과밀도가 줄고 CTA 클릭 유도가 개선된다.
- 태블릿/데스크톱에서 시각적 균형과 카드 완성도가 향상된다.
