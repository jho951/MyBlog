# ADR 006: 모바일 구독 섹션에서 mock 숨김 및 CTA 중앙 정렬

- 상태: 승인
- 날짜: 2026-02-28

## 배경

- 모바일 화면에서는 `spMock`가 정보 밀도를 높여 핵심 CTA 집중도를 낮출 수 있다.
- 요청사항으로 모바일에서 mock 영역 제거와 구독 버튼 중앙 정렬이 필요했다.

## 결정

- `max-width: 767px`에서 `.subscribePromo__mock`을 `display: none`으로 처리한다.
- 동일 구간에서 `.btn_subscription`을 블록 요소로 두고 좌우 자동 여백으로 중앙 정렬한다.

## 결과

- 모바일 구독 섹션이 간결해지고 CTA 집중도가 높아진다.
- 모바일에서 버튼 터치 동선이 명확해진다.
