# ADR 007: 구독 mock 영역 PC 전용 노출

- 상태: 승인
- 날짜: 2026-02-28

## 배경

- 구독 섹션의 mock 카드는 모바일뿐 아니라 태블릿에서도 정보 밀도를 높일 수 있다.
- 요구사항은 mock 영역을 PC에서만 노출하는 것이다.

## 결정

- `subscribePromo__mock`은 `max-width: 1024px` 구간에서 숨긴다.
- 결과적으로 mock 영역은 1025px 이상(PC)에서만 노출한다.

## 결과

- 모바일/태블릿에서 구독 섹션이 더 간결해지고 CTA 집중도가 높아진다.
- PC에서는 기존 mock 비주얼을 유지한다.
