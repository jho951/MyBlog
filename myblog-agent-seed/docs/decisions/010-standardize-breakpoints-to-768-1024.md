# ADR 010: 반응형 브레이크포인트를 768/1024로 표준화

- 상태: 승인
- 날짜: 2026-02-28

## 배경

- 스타일시트에 `max-width`, `1025px`, `768~1024` 혼합 구간이 공존해 기준이 분산되어 있었다.
- 모바일 퍼스트 기준에서 브레이크포인트를 단순화할 필요가 있었다.

## 결정

- `@media` 브레이크포인트는 `min-width: 768px`, `min-width: 1024px`만 사용한다.
- `max-width` 기반 미디어쿼리는 제거/변환한다.
- `min-width: 1025px`은 모두 `min-width: 1024px`로 통일한다.
- `min-width: 768px and max-width: 1024px` 형태는 `min-width: 768px` 기준으로 재정렬한다.

## 결과

- 반응형 기준이 두 개의 명확한 구간(768+, 1024+)으로 통일된다.
- 유지보수 시 브레이크포인트 판단 비용이 줄어든다.
