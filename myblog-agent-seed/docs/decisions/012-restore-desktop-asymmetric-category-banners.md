# ADR 012: PC에서 카테고리 비대칭 배너(catStack) 노출 복구

- 상태: 승인
- 날짜: 2026-03-01

## 배경

- 홈 카테고리 영역에서 `@media (min-width: 768px)` 오버라이드가 `catHub--mobile`을 강제 노출하고 `catStack`을 숨기고 있었다.
- 이 규칙이 PC(1024px+)에도 그대로 적용되어, 데스크톱 비대칭 배너 레이아웃(`catStack`)이 보이지 않았다.

## 결정

- `1024px+` 구간에서 `catHub--mobile`을 숨기고 `catStack`을 다시 노출한다.
- 태블릿(768~1024px)에서는 기존대로 `catHub--mobile` 스택형 노출을 유지한다.

## 결과

- PC에서 카테고리 영역이 원래 의도한 비대칭 배너 형태(`catStack`)로 복구된다.
- 태블릿과 PC의 카테고리 노출 책임이 다시 명확해진다.
