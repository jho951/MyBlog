# ADR 016: guestbook 코르크 보드 스타일 + 헤더 투명 처리

- 상태: 승인
- 날짜: 2026-03-01

## 배경

- guestbook은 기능적으로는 동작하지만 시각적 정체성이 약했고, 상단 안내 블록은 카드 배경이 있어 보드 콘셉트와 분리되어 보였다.
- 요구사항은 페이지를 코르크 보드 느낌으로 만들고, `GUESTBOOK` 안내 영역 배경은 투명하게 두는 것이다.

## 결정

- `#pageGuestbook .tt-guestbook__panel`에 코르크 텍스처(노이즈 점 패턴 + 브라운 그라데이션 + 인셋 그림자)를 적용한다.
- 상단 `guestbook-head`는 배경/보더/그림자를 제거해 투명 레이어로 둔다.
- 칩(`guestbook-chip`)은 완전 불투명 카드 대신 반투명 배경으로 조정해 헤더 투명 콘셉트를 유지한다.

## 결과

- guestbook 페이지가 보드형 UI로 일관성 있게 보인다.
- 상단 안내 텍스트 영역은 배경 없이 떠 있는 형태로 가독성과 레이아웃 의도가 명확해진다.
