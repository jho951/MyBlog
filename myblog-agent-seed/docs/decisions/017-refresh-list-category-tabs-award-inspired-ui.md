# ADR 017: 목록 카테고리 탭 UI를 수상작 레퍼런스 기반으로 리프레시

- 상태: 승인
- 날짜: 2026-03-01

## 배경

- 목록 페이지의 루트/하위 카테고리 탭이 기본형 스타일에 가까워, 현재 테마 품질 대비 UI 밀도가 낮아 보였다.
- 최근 수상작에서 자주 보이는 패턴(레이어드 글래스 패널, 강한 활성 상태 칩, 명확한 선택 포커스)을 반영해 탐색 인지성을 높일 필요가 있었다.
- 레퍼런스:
  - CSS Design Awards Winner of the Day `Victor Berbel Portfolio` (2025-11-14)
  - Awwwards 수상작 큐레이션의 네비게이션/칩 강조 트렌드

## 결정

- `#pageList .list-header`를 다층 그라데이션 + 소프트 노이즈 + 블러 패널로 재설계한다.
- 루트 카테고리 탭은 Pill 형태를 강화하고, 활성 상태에 고대비 배경/글로우/점 인디케이터를 추가한다.
- 하위 카테고리 쉘/탭도 동일한 계층 언어로 통일해 루트-하위 탐색 흐름을 일관화한다.

## 결과

- 카테고리 선택 상태가 한눈에 구분되고, 탭 인터랙션의 시각 피드백이 강화된다.
- 목록 페이지 상단 탐색 UI가 홈/테마 품질과 균형을 맞추게 된다.
