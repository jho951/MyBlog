# ADR 018: guestbook 헤더 제거 및 전체 코르크 보드화

- 상태: 승인
- 날짜: 2026-03-01

## 배경

- guestbook 상단의 안내 헤더(`GUESTBOOK`, 제목/설명/칩)가 별도 블록으로 존재해, 코르크 보드 콘셉트가 분리되어 보였다.
- 요구사항은 해당 안내 영역을 제거하고 코르크 보드가 화면에서 하나의 큰 보드처럼 보이게 하는 것이다.

## 결정

- `<s_guest>` 영역에서 `guestbook-head` 마크업을 제거한다.
- 코르크 텍스처 배경은 `.tt-guestbook__panel`이 아니라 `#pageGuestbook .page-wrap`에 적용한다.
- 내부 패널(`.tt-guestbook__panel`)은 투명 컨테이너로 단순화해 보드 일체감을 유지한다.

## 결과

- guestbook이 단일 코르크 보드 레이어로 인지된다.
- 불필요한 상단 설명 블록이 제거되어 메시지 작성/읽기 흐름이 더 간결해진다.
