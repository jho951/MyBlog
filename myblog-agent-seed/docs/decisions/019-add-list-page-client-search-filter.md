# ADR 019: 목록 페이지 클라이언트 검색 필터 추가

- 상태: 승인
- 날짜: 2026-03-01

## 배경

- 목록 페이지에서 카테고리 탭만으로는 원하는 글을 빠르게 찾기 어려운 경우가 있었다.
- 검색 기능 요청에 따라 페이지 이탈 없이 즉시 필터링되는 경량 검색이 필요했다.

## 결정

- `list-search-filter.js`를 추가해 `#pageList`에서 제목/카테고리 텍스트 기반 클라이언트 필터를 제공한다.
- 검색 입력, 지우기 버튼, 결과 개수 메타 정보를 목록 헤더에 배치한다.
- 서버 검색/URL 변경 없이 현재 렌더링된 카드 DOM만 대상으로 동작한다.

## 결과

- 목록 페이지에서 키워드 입력 즉시 결과를 좁힐 수 있다.
- 카테고리 탭 탐색과 검색 필터를 병행해 글 탐색 효율이 개선된다.
