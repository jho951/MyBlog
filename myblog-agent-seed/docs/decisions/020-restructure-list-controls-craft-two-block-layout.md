# ADR 020: 목록 컨트롤을 craft 2블록 레이아웃으로 재구성

- 상태: 승인
- 날짜: 2026-03-01

## 배경

- 기존 목록 헤더는 요소가 일렬로 이어진 구조라 시선 분산이 크고, 탐색 우선순위가 약했다.
- 사용자 요구는 디자인 구조 자체를 바꾸는 것이며, 탭 UI/UX를 더 높은 완성도로 끌어올리는 것이다.

## 결정

- 목록 헤더를 `list-control-main(타이틀/뷰 토글/검색)`과 `list-control-tabs(루트 카테고리)` 2블록 구조로 분리한다.
- 1024px+에서 2컬럼 레이아웃을 적용해 정보 블록과 탐색 블록을 병렬 배치한다.
- 루트/하위 탭은 craft 톤(웜 그라데이션, 깊은 active state, 명확한 상태 대비)으로 통일한다.

## 결과

- 목록 진입 시 정보(현재 뷰, 검색)와 탐색(카테고리)의 역할이 명확해진다.
- 모바일에서는 단일 흐름, 데스크톱에서는 분리된 탐색 구조로 가독성과 조작성이 함께 개선된다.
