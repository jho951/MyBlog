# ADR 021: 목록 페이지를 사이드 트리 탭 + 미니멀 상단 툴바로 재구성

- 상태: 승인
- 날짜: 2026-03-01

## 배경

- 기존 목록 상단은 탭/검색/타이틀이 한 블록에 혼합되어 정보 구조가 복잡했다.
- 요구사항은 상단에는 `뷰 토글 + 검색 버튼`만 두고, 카테고리 탭은 좌측 트리 노드 형태로 분리하는 것이다.

## 결정

- `list-header`는 툴바 역할만 수행하고, `list-layout` 내 `list-sidebar`에 루트/하위 탭을 배치한다.
- 루트/하위 탭은 세로 트리 노드 UI로 구성한다.
- 검색 입력은 상단 버튼으로 여닫는 패널(`ttListSearchPanel`)로 전환한다.

## 결과

- 상단 조작과 좌측 탐색의 역할이 분리되어 탐색 구조가 명확해진다.
- 데스크톱에서 사이드 트리 탐색 효율이 개선되고, 모바일에서는 단일 세로 흐름으로 유지된다.
