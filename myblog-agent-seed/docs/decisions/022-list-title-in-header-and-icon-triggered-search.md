# ADR 022: list-title 상단 배치 + 아이콘 트리거 검색 패널

- 상태: 승인
- 날짜: 2026-03-01

## 배경

- 목록 페이지에서 `list-title`이 콘텐츠 영역에 있어 헤더 역할이 약했고, 툴바는 배경이 과해 상단 정보 밀도가 높았다.
- 요구사항은 상단 우측에 토글 버튼을 배치하고, 그 좌측 검색 아이콘 클릭 시 입력창이 나타나는 형태다.

## 결정

- `list-title`을 `list-header`의 `list-header-top`으로 이동한다.
- `list-toolbar`는 배경 없는 정렬 컨테이너로 단순화한다.
- 툴바 구성은 `검색 아이콘 버튼 → 리스트/카드 토글` 순서로 배치한다.
- 검색 입력은 기본 숨김 패널(`ttListSearchPanel`)로 두고, 아이콘 클릭 시 표시한다.

## 결과

- 상단 정보 구조가 `제목(좌) / 액션(우)`로 단순해져 가독성이 개선된다.
- 검색 입력은 필요할 때만 노출되어 시각적 잡음이 줄어든다.
