# ADR 023: 목록 툴바 검색을 인라인 슬라이드 확장으로 전환

- 상태: 승인
- 날짜: 2026-03-01

## 배경

목록 페이지 상단 툴바는 `검색 아이콘 + 리스트/카드 토글`만 노출하기로 정리되었지만, 기존 구현은 검색 아이콘 클릭 시 툴바 하단에 별도 패널이 열리는 구조였다.  
요구사항은 아이콘 인접 위치에서 입력창이 직접 확장되는 형태다.

## 결정

- `ttListSearchPanel`을 툴바 내부(검색 버튼 오른쪽) 인라인 컨테이너로 이동한다.
- 검색 패널 열림/닫힘은 `hidden` 속성 토글 대신 `is-open` 클래스 토글로 제어한다.
- 애니메이션은 `max-width`, `opacity`, `transform` 조합으로 구현한다.
- 결과 카운트(`ttListSearchMeta`)는 헤더 하단 텍스트로 유지해 검색 피드백을 제공한다.

## 결과

- 검색 입력이 툴바 흐름을 유지한 채 자연스럽게 슬라이드 인/아웃된다.
- UI 구조가 단순해지고, 검색 컨트롤과 보기 토글의 시각적 결합도가 올라간다.
- `list-search-filter.js`는 클래스 기반 상태 관리로 전환되어 CSS 애니메이션과 충돌이 줄어든다.
