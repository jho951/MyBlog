# ADR 024: 목록 사이드바 루트-서브 드롭다운 트리 구조

- 상태: 승인
- 날짜: 2026-03-01

## 배경

목록 페이지는 좌측 사이드 트리 구조를 사용하고 있으나, 하위 카테고리가 루트와 분리된 별도 섹션으로 렌더링되어 루트 기준 계층 관계가 약했다.  
요구사항은 루트 카테고리 우측 화살표를 눌러 해당 루트의 하위 카테고리를 바로 펼치는 드롭다운 구조다.

## 결정

- `ttListCategoryTabs` 내부를 루트 노드 단위(`list-category-node`)로 재구성한다.
- 하위 카테고리가 있는 루트는 우측에 화살표 버튼(`list-category-expand`)을 배치한다.
- 화살표 버튼은 `aria-expanded`/`aria-controls`를 사용해 접근 가능한 접기/펼치기 상태를 제공한다.
- 활성 루트 카테고리는 초기 로드 시 자동 확장한다.
- 기존 별도 하위 카테고리 섹션(`ttListSubcategoryShell`)은 제거한다.

## 결과

- 루트-서브 계층이 한 덩어리로 보이면서 탐색 흐름이 직관적으로 개선된다.
- 사용자는 루트 이동과 하위 탐색을 동일 위치에서 처리할 수 있다.
- 토글 상태가 JS 클래스(`is-expanded`)와 aria 속성으로 동기화되어 상태 추적이 명확해진다.
