# ADR 025: 사이드바 드롭다운에서 활성 서브카테고리만 노출

- 상태: 승인
- 날짜: 2026-03-01

## 배경

사이드바 트리를 루트 기준 드롭다운으로 전환한 이후, 서브카테고리 페이지에서도 해당 루트의 서브카테고리 전체가 노출되어 정보 밀도가 높았다.  
요구사항은 클릭 후 활성된 서브카테고리만 보이도록 단순화하는 것이다.

## 결정

- `list-subcategory-tabs.js`의 서브카테고리 렌더링 로직에 `activeOnly` 조건을 추가한다.
- 현재 경로가 특정 서브카테고리인 경우(`activeSubSlug` 존재), 해당 루트 드롭다운에는 활성 서브카테고리 1개만 렌더링한다.
- 이 경우 `전체` 링크와 비활성 서브카테고리 항목은 렌더링하지 않는다.

## 결과

- 서브카테고리 페이지의 사이드바가 간결해지고 현재 위치 인지가 더 빨라진다.
- 루트/서브의 활성 상태 표시는 유지하면서 시각적 노이즈를 줄일 수 있다.
