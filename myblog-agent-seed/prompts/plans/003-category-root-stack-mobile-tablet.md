# 카테고리 루트 스택 통일 작업 프롬프트

목표:
- 홈 카테고리 영역을 모바일/태블릿/PC에서 동일한 정보 구조(루트 카테고리만)로 맞춘다.

작업 지시:
- `src/pages/main.html`에서 카테고리 섹션의 접근성 라벨을 root-only 구조에 맞게 정리한다.
- `src/style/style.css`에서 `catHub--mobile` 노출을 비활성화한다.
- `catStack`이 모바일부터 기본 노출(`display:flex`)되도록 조정한다.
- 1025px 이상 데스크톱 구간의 기존 2열 레이아웃은 유지한다.

검증 기준:
- 모바일(<=767px): 루트 카테고리 4개가 세로로 노출된다.
- 태블릿(768~1024px): 루트 카테고리만 세로로 노출된다.
- 데스크톱(>=1025px): 기존 2열 `catStack` 레이아웃이 유지된다.
