# 구독 섹션 반응형 디자인 정리

목표:
- 홈 `subscribePromo` 섹션을 모바일/태블릿/데스크톱에서 균형 있게 보이도록 개선한다.

작업 지시:
- `src/style/style.css`의 `subscribePromo/spMock` 블록을 모바일 우선으로 재정비한다.
- 모바일은 1열(카피 우선), 태블릿 이상은 2열로 확장한다.
- 구독 버튼(`btn_subscription`)의 크기/폭/라운드/그림자를 조정해 CTA를 강화한다.
- `main.html`의 버튼 클래스에서 `#subscribe` 토큰을 제거한다.

검증 기준:
- 모바일에서 텍스트/버튼이 겹치지 않고 읽기 순서가 자연스럽다.
- 태블릿/PC에서 목업 카드와 카피 영역이 균형 있게 배치된다.
- 구독 버튼이 화면 폭에 맞춰 적절히 확장/축소된다.
