# 목록 페이지 검색 필터 추가

목표:
- 목록 페이지에서 제목/카테고리 기준으로 즉시 필터링 가능한 검색 기능을 제공한다.

작업 지시:
- `src/pages/main.html`의 목록 헤더에 검색 입력/지우기 버튼/결과 메타 영역을 추가한다.
- `src/function/list-search-filter.js`를 생성해 카드 DOM 필터 로직을 구현한다.
- `src/style/style.css`에 검색 UI 스타일을 추가해 탭 디자인과 동일한 톤으로 맞춘다.

검증 기준:
- 검색어 입력 시 목록 카드가 즉시 필터링된다.
- 지우기 버튼으로 원복 가능하다.
- 결과 개수(`검색 결과 x / total`)가 실시간으로 갱신된다.
