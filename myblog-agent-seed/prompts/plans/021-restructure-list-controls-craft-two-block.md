# 목록 컨트롤 구조 재구성 (craft 2블록)

목표:
- 목록 페이지 상단 컨트롤을 `메인 정보(타이틀/검색)`와 `루트 탭 탐색`으로 분리해 구조적으로 개선한다.

작업 지시:
- `src/pages/main.html`의 `list-header` 내부를 `list-control-main`, `list-control-tabs` 구조로 재배치한다.
- `src/style/style.css`에서 1024px+ 2컬럼 레이아웃을 적용하고, 모바일/태블릿은 단일 흐름으로 유지한다.
- 루트/하위 탭 active/hover 대비를 craft 톤으로 강화한다.

검증 기준:
- 모바일에서는 블록이 위아래 순서로 자연스럽게 노출된다.
- 1024px+에서는 메인 블록과 루트 탭 블록이 병렬 배치된다.
- 상/하위 탭 선택 상태가 비선택 상태와 명확히 구분된다.
