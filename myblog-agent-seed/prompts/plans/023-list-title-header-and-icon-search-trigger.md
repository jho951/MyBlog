# 목록 헤더 타이틀 배치 + 아이콘 검색 트리거

목표:
- `list-title`을 `list-header` 상단에 배치한다.
- 툴바는 배경 없이 우측 액션 영역으로 유지하고, 검색 아이콘 클릭 시 입력 패널이 열리게 한다.

작업 지시:
- `src/pages/main.html`에서 `list-title`을 `list-header-top`으로 이동하고 콘텐츠 영역의 중복 타이틀을 제거한다.
- `list-toolbar`의 버튼 순서를 `검색 아이콘 -> 뷰 토글`로 정렬한다.
- `src/style/style.css`에서 툴바 배경을 제거하고 아이콘 버튼/토글 버튼을 동일 컴포넌트 톤으로 정리한다.

검증 기준:
- 상단 좌측에 `list-title`, 우측에 검색 아이콘과 뷰 토글이 보인다.
- 검색 아이콘 클릭 시 `ttListSearchPanel`이 나타난다.
- 목록 본문에 중복 타이틀이 노출되지 않는다.
