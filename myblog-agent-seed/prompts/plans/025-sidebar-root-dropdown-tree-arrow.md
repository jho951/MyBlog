# Plan Prompt 025

목표: 목록 페이지 좌측 사이드바에서 하위 카테고리를 루트 카테고리별 드롭다운 트리로 노출한다.

요구사항:
- 루트 카테고리 행 우측에 화살표 토글 버튼을 배치한다.
- 화살표 클릭 시 해당 루트의 하위 카테고리가 루트 바로 아래에 드롭다운된다.
- 활성 루트 카테고리는 초기 진입 시 자동 확장한다.
- `aria-expanded`, `aria-controls`를 사용해 접근성 상태를 제공한다.
- 루트 링크 클릭은 기존처럼 카테고리 페이지 이동을 유지한다.

체크리스트:
1. 루트 우측 화살표 클릭 시 하위 카테고리가 열리고 다시 클릭 시 닫히는가?
2. 활성 루트에서 초기 로드 시 하위 목록이 열려 있는가?
3. 루트 링크 클릭 시 드롭다운 토글이 아니라 실제 라우팅이 동작하는가?
4. 모바일/태블릿/PC에서 트리 들여쓰기와 화살표 위치가 깨지지 않는가?
