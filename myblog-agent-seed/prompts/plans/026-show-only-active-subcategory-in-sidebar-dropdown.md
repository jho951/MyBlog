# Plan Prompt 026

목표: 서브카테고리 페이지에서 사이드바 드롭다운 목록을 최소화해, 활성된 서브카테고리 항목만 노출한다.

요구사항:
- 현재 경로가 `/category/{root}/{sub}` 형태일 때, 해당 루트 드롭다운에는 활성 `{sub}`만 렌더링한다.
- 비활성 서브카테고리 및 `전체` 항목은 숨긴다.
- 루트 카테고리 페이지(`/category/{root}`)에서는 기존처럼 `전체 + 서브카테고리 목록`을 유지한다.
- 활성 상태 클래스/aria 속성은 기존 규칙을 유지한다.

체크리스트:
1. 서브카테고리 페이지에서 드롭다운에 활성 서브카테고리 1개만 보이는가?
2. 루트 카테고리 페이지에서는 서브카테고리 전체가 보이는가?
3. 활성 루트 자동 확장 및 화살표 상태가 정상 동기화되는가?
